  • XP、Windows Server 2003、win8等系统访问：

  1. 关闭445,139等危险端口

  2. 创建互斥体:

因为加密的特殊性,加密器会在启动时检测是否有j加密程序存在,防止互相干扰,因此创建互斥器MsWinZonesCacheCounterMutexA来让病毒自动退出,从而防止病毒加密文件
